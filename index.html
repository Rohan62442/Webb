Create & manage Web Applications

Webb is a script for managing web applications based on GNOME's Epiphany web browser, which is now simply known as Web.

To install Webb, just open a Terminal and run this command:

curl https://rohan62442.github.io/Webb/webb-updater -o webb-updater && chmod +x webb-updater && ./webb-updater
